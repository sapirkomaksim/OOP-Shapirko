## Лабораторна Робота №21
### Виконав: Шапірко Максим Вікторович ІПЗ 3/1

### Тема: OCP: гнучкі алгоритми розрахунку (Factory/Strategy).
Мета: Застосувати принцип відкритості/закритості (OCP) для створення гнучкої системи
розрахунків за допомогою патернів Factory Method та Strategy, забезпечивши можливість легкого
додавання нових алгоритмів без зміни існуючого коду.

У ході лабораторної роботи було закріплено теми «Принципи об'єктно-орієнтованого дизайну SOLID (SRP, OCP) та патерни проектування».
Проведено аналіз архітектури open-source проєкту на C# (nopCommerce), де ідентифіковано випадки дотримання SRP в інфраструктурних сервісах та порушення у складних бізнес-класах. У практичній частині створено консольний додаток, що імітує систему доставки. Продемонстровано реалізацію патерну проектування «Стратегія» (Strategy) через інтерфейс IShippingStrategy та використання «Фабричного методу» (Factory Method) для динамічного вибору алгоритму. Програма виконує розрахунок вартості доставки для різних умов (Standard, Express, International), забезпечуючи виконання принципу OCP: додавання нової логіки (NightShippingStrategy) здійснено без втручання в основний клас сервісу DeliveryService. Таким чином закріплено навички проектування гнучкої архітектури програмного забезпечення, рефакторингу коду та створення систем, які відкриті для розширення, але закриті для модифікації.
